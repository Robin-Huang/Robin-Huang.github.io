Android Studio + OpenCV3 安裝

發表於 2019-07-14 更新於 2019-07-19 分類於 Android 評論數：

透過 Android Studio 結合 OpenCV 做影像處理和電腦視覺的 APP 應用

軟體需求

Android Studio
OpenCV 3.X (Android)

安裝流程

下載 Android Studio


下載 OpenCV 3.X Android 安裝包


解壓縮 OpenCV 3.x Android


安裝 Android Studio，並設定好 Android SDK、NDK、Build-Tools 等套件

Android Studio 設定

開啟 Android Studio，選擇建立新專案


選擇 Empty Project


設定 Project 名稱、路徑


File -> New -> Import Module


指定 OpenCV Android Module 路徑，會在解壓縮資料夾路徑下的 ./sdk/java/ ，指定完會自動偵測模組名稱


模組設定完後可能會跳出以下 Error ，表示 minSdkVersion 和 targetSdkVersion 不應該被宣告在 manifest 檔案裡


進入 ./openCVLibrary3XX/src/main/AndroidManifest.xml 中，將紅框中的設定刪掉


開啟 openCVLibrary3XX 的 build.gradle ，把 compileSdkVersion、buildToolsVersion、minSdkVersion、targetSdkVersion 設定與 app 中的 build.gradle 一致


建立一個 jni 資料夾



勾選 Change Folder Location ，並將資料夾名稱改為 jniLibs ，Finish



將 OpenCV 3.XX Android 的 libs 檔複製到剛剛建立的 jniLibs 資料夾




開啟 Project Structure，對 app 新增 Module Dependency



選擇 openCVLibrary3XX


使用 OpenCV
