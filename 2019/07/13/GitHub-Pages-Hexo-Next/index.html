GitHub Pages + Hexo + NexT 建立個人網頁

透過 Github Pages 和 Hexo 建立自己的個人網頁，並使用 Next 主題

軟體需求

Git
Node.js
Hexo

安裝流程

先將 Git 和 Node.js 安裝好，利用 Node.js 的 npm 管理工具安裝 Hexo

初始化 Hexo

啟動 local 端的 server

使用 NexT 主題

在 theme 資料夾下會存放網站的主題，Hexo 預設使用 landscape 這個主題，我比較喜歡 NexT 這個主題

用 Git 安裝 NexT

修改 ./_config.yml 中的主題設定

再啟動一次 server 就會看到主題切換成 NexT 了

語言設定

再 ./_config.yml 中設定語言

程式碼主題

NexT 中有 5 種程式碼主題可以選擇，normal、night、night blue、night bright 和 night eighties

Sheme 設定

NexT 提供 4 種格式，Muse、Mist、Pisces 和 Gemini，可以依照自己的喜好做選擇

作者設定

在 ./_config.yml 中設定標題、副標題、描述、關鍵字和作者名字

社群連結

在 ./theme/next/_config.yml 中可以設定 Github、G-mail、Twitter 等連結

建立文章

所有文章都會在 ./source/_posts 路徑下。可以看到裡面已有一篇範例文章 hello-world.md

在 ./_config.yml 設定 post_asset_folder ，每當新增一篇文章時會自動在 ./source/_posts 建立一個跟文章同名的資料夾放圖片

新增文章
